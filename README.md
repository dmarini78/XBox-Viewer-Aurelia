# XBox-Viewer-Aurelia
A quick and dirty first app written in Aurelia

Install the tooling packages. 

 ```shell
  npm install
  ```

  ```shell
  npm install -g gulp
  ```
  
  ```shell
  npm install -g jspm
  ```

Install the pre-requisite packages
  ```shell
  jspm install -y

then just run

```shell
gulp watch

and browse to port 9000
